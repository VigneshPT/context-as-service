What is Contextify?

Contextify is a cloud based Context As A Service, takes several inputs from the user's mobile device and describes the context of the environment the user is in. For example, the service detects when the user is on-the-go based on the speed and delivers context to the consuming applications relatively. Contextify provides an API to developers in all major platforms, which allows them to leverage the service in their application and provide valuable content.

Why Contextify?

Any application can use Contextify API as a library and can utilize the user's context based keywords for its business logic and can deliver personalized service to its users.

How It Works?

An application needs to personalize the content it delivers to its users based on his context of the current location he is in. It simply uses our Contextify library and calls our API.Contextify API gets several inputs such as location, audio inputs, calendar events and even the moving speed of the user and generates a set of keywords related to the context of his currrent environment and delivers it back to the application.
